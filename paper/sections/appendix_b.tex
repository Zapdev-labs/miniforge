\subsection{Configuration File Format}

Configuration can be loaded from YAML files:

\textbf{Example Configuration Structure:}
\begin{itemize}
    \item Hardware limits: Memory allocation and threading
    \item Model settings: Context size and quantization
    \item Performance: Optimization features and backend
    \item Defaults: Generation parameter defaults
    \item Features: Enable/disable capabilities
\end{itemize}

\subsection{Backend Configuration}

Backend-specific settings for fine-tuning:

\subsubsection{llama.cpp Backend}

\textbf{Key Parameters:}
\begin{itemize}
    \item \texttt{n\_ctx:} 194560 (context window)
    \item \texttt{n\_threads:} 8 (CPU threads)
    \item \texttt{n\_batch:} 2048 (batch size)
    \item \texttt{n\_ubatch:} 512 (micro-batch)
    \item \texttt{cache\_type\_k/v:} turbo3 (KV cache compression)
    \item \texttt{flash\_attn:} true (Flash Attention)
    \item \texttt{use\_mmap:} true (memory mapping)
    \item \texttt{use\_mlock:} false (WSL2 limitation)
    \item \texttt{rope\_freq\_base:} 10000.0 (RoPE base frequency)
    \item \texttt{rope\_freq\_scale:} 1.0 (RoPE scaling)
\end{itemize}

\subsubsection{Transformers Backend}

\textbf{Key Parameters:}
\begin{itemize}
    \item \texttt{torch\_dtype:} float16 (half precision)
    \item \texttt{device\_map:} auto (automatic placement)
    \item \texttt{low\_cpu\_mem\_usage:} true (reduce memory)
    \item \texttt{attn\_implementation:} eager (attention backend)
\end{itemize}

\subsection{Context Window Sizing}

Recommended context sizes by task:

\begin{itemize}
    \item \textbf{Chat:} 4K-8K sufficient
    \item \textbf{Document Q\&A:} 16K-32K
    \item \textbf{Long document analysis:} 64K-128K
    \item \textbf{Book-length processing:} 192K (full)
\end{itemize}

\subsection{Performance Tuning}

\subsubsection{Thread Count}

Recommended thread counts by CPU:

\begin{itemize}
    \item 4-core: 4 threads
    \item 6-core: 6 threads
    \item 8-core: 8 threads
    \item 12+ core: 8-12 threads (diminishing returns)
\end{itemize}

\subsubsection{Batch Size}

Batch size impacts:

\begin{itemize}
    \item Larger batches: Better throughput, higher memory
    \item Smaller batches: Lower latency, lower memory
    \item Default 2048 balances both
\end{itemize}
