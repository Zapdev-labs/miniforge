\subsection{Installation}

\subsubsection{Quick Install}

Standard installation from PyPI:
\begin{itemize}
    \item Run \texttt{pip install miniforge}
    \item Requires Python 3.10+
    \item Optional dependencies: \texttt{[llama-cpp]}, \texttt{[transformers]}, \texttt{[all]}
\end{itemize}

\subsubsection{Development Install}

For contributors and developers:
\begin{enumerate}
    \item Clone repository: \texttt{git clone https://github.com/Zapdev-labs/miniforge.git}
    \item Navigate to directory: \texttt{cd miniforge}
    \item Install with dev dependencies: \texttt{uv pip install -e ".[all]"}
    \item Run tests: \texttt{pytest tests/}
\end{enumerate}

\subsection{Quick Start Usage}

\textbf{Basic Workflow:}
\begin{enumerate}
    \item \textbf{Import:} Load \texttt{Miniforge} class from package
    \item \textbf{Initialize:} Load model with \texttt{from\_pretrained()}
    \item \textbf{Configure:} Set quantization, context, and backend
    \item \textbf{Generate:} Use \texttt{chat()} or \texttt{generate()} methods
    \item \textbf{Stream:} Enable streaming for real-time output
    \item \textbf{Cleanup:} Release resources with \texttt{cleanup()}
\end{enumerate}

\textbf{Key Features Demonstrated:}
\begin{itemize}
    \item Simple chat interface with system prompts
    \item Streaming token generation
    \item Automatic resource management
    \item Async/await pattern throughout
\end{itemize}
